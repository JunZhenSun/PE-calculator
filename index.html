计算逻辑
v1 简单版
PE = Σ (1+gₜ)ᵗ
直接累加名义利润，不做折现
v3 修正版
PE = Σ (1+gₜ)ᵗ / (1+d)ᵗ⁺¹
每年净利润按折现率折算为现值后累加
gₜ = 第t年增速（支持恒定/衰减/分阶段） · d = 折现率
回本定义：累计净利润（含留存再投资部分）≥ 买入价格
v3 vs v2 的区别：移除了ROIC再投资扣除。理由：净利润无论分红还是留存，都属于股东——留存部分增加了公司净资产（即你持有的权益），不应从"回本"中扣除。唯一保留的修正是折现（货币时间价值）和增速衰减（均值回归的现实建模）。
